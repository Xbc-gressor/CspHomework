JDK的简介，卸载和安装过程

JDK的简介，卸载和安装过程

JDK JRE JVM

  • JDK：Java Development Kit

  • JRE：Java Runtime Environment

  • JVM：Java Virtual Machine

卸载JDK

  1. 控制面板-程序和功能-卸载jdk环境
  2. 删除JAVA_HOME（环境变量）
  3. 删除系统变量path下关于Java的目录
  4. 在cmd里输入java -version找不到了就成功了

安装JDK

  1. 百度搜索JDK8

  2. 同意协议

  3. 下载电脑对应版本

  4. 双击安装JDK

  5. 记住安装的路径

  6. 配置环境变量

    6.1 我的电脑->右键->属性->高级系统设置->环境变量

    6.2 环境变量->系统变量->新建JAVA_HOME (jdk的安装路径)

    JAVA_HOME：C:\Program Files\Java\jdk1.8.0_181

    6.3 配置系统变量里的PATH变量

    如果是win10系统，直接新建两个，分别填%JAVA_HOME%\bin

    和%JAVA_HOME%\jre\bin

    如果是win7系统，在原来Path后面追加%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin，注意一下如果原来Path最后没有;的话需要先加一个;

    6.4 配置系统变量里的CLASSPATH变量(新建)

    CLASSPATH：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar

  7. 测试JDK是否安装成功

    windows+R打开运行，输入cmd打开cmd窗口，输入java -version，如果能显示JDK版本就说明安装成功了
